Quick-Start Guide

  1. Drag a MadCat exported XML, or media files (or folders containing them) into Spectator.

    Skip this step if you are opening the files you want straight from MadCat.

  2. Once all tracks have been loaded, you can control playback for all tracks with the bottom controls, or individual tracks using the side control panels.

    If any of the tracks are video files, clicking play will open a video window to display them.

  3. You can toggle all tracks to display between Spectrogram or Waveform mode, toggle them individually and also change the axis scales.

Compare Media files in Spectator

  1. To open files in Spectator, you can load either a selection of media files or folders containing them, an XML file that was either previously created by Spectator or exported from MadCat, or just open Spectator from MadCat with the desired files (see the MadCat manual for how to do this). XMLs or media files/folders can be dragged into Spectator (attempting to drag in both will have Spectator load the media files and ignore the XML). Folders will be recursively searched for media files. XML files can also be loaded via the File>Open dropdown option.

  2. Spectator will then load the selected media files and synchronise them using their audio. Each file will display as its own track as a Spectrogram or Waveform display with accompanying track controls for muting, soloing, changing the track volume and changing the stereo output. Track information can be seen by hovering the mouse over the Spectrogram/Waveform display, and the display can be toggled from Spectrogram/Waveform mode by the lower right box that appears when hovering over the track display. The top X-Axis is the time, whilst the Y-Axis will be either frequency (if the track is in Spectrogram display mode) or decibel level (if in Waveform display mode).

  3. Once all desired tracks are loaded, playback can be made to start via the Play button beneath all the tracks. If any of the tracks are video files, this will open up a Video Window which will have a section to display the video of each of the video files being played. Playback can be paused using the Pause button and stopped with the Stop button, which will also close the Video Window if open. You can also seek to points within the tracks via clicking on them at the position you wish to go to, when doing this the time position you have set will appear briefly left of the Play button.

  4. All tracks can be muted or unmuted via the lower left Mute All button, and also have their Solo state set or unset via the Solo All button. The display of all tracks can be toggled to Waveform/Spectrogram mode via either the lower left Toggle Waveform/Spectrogram View All Tracks button or the View>Spectrogram and View>Waveform options. The lower right buttons allow the opening of a slider to adjust the Y-Axis scale, the increase of the X-Axis scale and the decrease of the X-Axis scale respectively. The axis scales can also be modified via the View dropdown menu.

  5. The Preferences dropdown menu allows you to alter how Spectator should behave:

    • Audio Hardware will allow you to set what audio hardware Spectator should output to.
    • Zoom Level will allow you to save the current Axis scales, or reset them back to their default size.
    • Default View will allow you to change whether Spectator should open in either Spectrogram or Waveform display mode by default, and will also change all current tracks to that display mode when selected.

  6. Spectator can save the set of files it has opened and their synchronised states through an XML document via File>Save and File>Save as…. This XML can then be re-opened into Spectator or imported into Adobe's Audition and Premiere programs as projects. Note that the media files used cannot be moved or renamed between the Spectator XML export and either importing into an Adobe program or re-opening in Spectator, and also that importing an XML that includes video files into Adobe Audition is not supported. See the “Import as Adobe projects guide” for more details on this.

  7. Spectator can also export a multichannel audio WAV file via File>Export>Multichannel wave file dropdown options. Each channel will be the audio of each loaded track. If the File>Export>Multichannel wave file with channel map… is selected, a text file with the same name as the WAV file will be created alongside that will give information on each channel.

Minimum Specifications:

  • Microsoft Windows 7 or 8, 64-bit editions.

  • 4 GB of RAM.
